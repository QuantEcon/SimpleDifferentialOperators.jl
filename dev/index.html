Installation

Installation

To install, run

] add SimpleDifferentialOperators

Note that this requires Julia 1.0 or later.

For the complete derivations of the objects we return, see the PDF.

Usage

This package provides discretized differential operators of first order and second order under reflecting boundary conditions.

Formula

Under $P$-length of grids on $v$ with end points $\underline{z} < \bar{z}$ with reflecting barrier conditions of

\[\begin{align} \xi v(\underline{z}) + \nabla v(\underline{z} ) &= 0\\ \xi v(\bar{z}) + \nabla v(\bar{z}) &= 0 \end{align}\]

We use the following discretization schemes:

\[L_1^{-} \equiv \frac{1}{\Delta}\begin{pmatrix} 1 - (1 + \xi \Delta) &0&0&\dots&0&0&0\\ -1&1&0&\dots&0&0&0\\ \vdots&\vdots&\vdots&\ddots&\vdots&\vdots&\vdots\\ 0&0&0&\dots&-1&1&0\\ 0&0&0&\cdots&0&-1&1 \end{pmatrix}_{P\times P}\]
\[L_1^{+} \equiv \frac{1}{\Delta}\begin{pmatrix} -1&1&0&\dots&0&0&0\\ 0&-1&1&\dots&0&0&0\\ \vdots&\vdots&\vdots&\ddots&\vdots&\vdots&\vdots\\ 0&0&0&\dots&0&-1&1\\ 0&0&0&\cdots&0&0&-1+(1-\xi \Delta) \end{pmatrix}_{P\times P}\label{eq:L-1-plus-regular} \\\]
\[L_2 \equiv \frac{1}{\Delta^2}\begin{pmatrix} -2 + (1 + \xi\Delta) &1&0&\dots&0&0&0\\ 1&-2&1&\dots&0&0&0\\ \vdots&\vdots&\vdots&\ddots&\vdots&\vdots&\vdots\\ 0&0&0&\dots&1&-2&1\\ 0&0&0&\cdots&0&1&-2 + (1- \xi\Delta) \end{pmatrix}_{P\times P}\label{eq:L-2-regular}\]

which represent the backward first order, foward first order, and central second order differential operators respectively.

Derivation, including formula for irregular grids, can be found here.
